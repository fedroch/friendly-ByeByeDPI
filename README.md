Приложение для Android, которое локально запускает ByeDPI и перенаправляет весь трафик через него.

Для стабильной работы может потребоваться изменить настройки. Подробнее о различных настройках можно прочитать в [документации ByeDPI](https://github.com/hufrea/byedpi/blob/v0.13/README.md).

Приложение не является VPN. Оно использует VPN-режим на Android для перенаправления трафика, но не передает ничего на удаленный сервер. Оно не шифрует трафик и не скрывает ваш IP-адрес.

Приложения является форком [ByeByeDPIAndroid](https://github.com/romanvht/ByeByeDPI)

### Что это? 
Приложение для обхода DPI (deep packet inspection) на android 

### Как это? 
Сейчас будет небольшая историческа справка буквально на пару минут... Чтобы определить, куда именно ваще устройство отправляет пакеты DPI проверяет пакет clienthello (в нем устройство и сервер договариваются о шифровании) и сверяет его содержимое со списком запрещенных доменов. Стратегия обхода создается комбинированием нескольких примитивных методов обмана DPI.

### Как пользоваться?
При первом запуске вам предложат протестировать стратегии обхода по умолчанию. После того как тесты завершились, выбираете стратегию, у которой открылось наибольшее число доменов, и нажимаете "приминить и запустить", выходите в главное меню и запускаете. Если стратегия обхода перестала работать, запустите тесты ещё раз (это можно сделать в меню настроек)

### Это впн? 
Нет. Android не позволяет обычному приложению получить доступ к трафику без root прав, поэтому приходится притворяться VPNом

### Почему не существует универсальной стратегии обхода?
В РКН все-таки не глупые люди работают... Разные провайдеры используют разные методы борьбы с обходом.

### Это замена VPN? 
Нет. В отличие от впн это приложение не даст вам возможность пользоваться полностью заблокированными ресурсами (вообще-то может но для этого нужно долго подбирать стратегию).

### Возможности
* Автозапуск сервиса при старте устройства
* Сохранение списков параметров командной строки
* Улучшена совместимость с Android TV/BOX
* Раздельное туннелирование приложений
* Импорт/экспорт настроек

### Использование
* Для работы автозапуска активируйте пункт в настройках.
* Рекомендуется подключится один раз к VPN, чтобы принять запрос.
* После этого, при загрузке устройства, приложение автоматически запустит сервис в зависимости от настроек (VPN/Proxy)
* Комплексная инструкция от комьюнити [ByeByeDPI-Manual](https://github.com/BDManual/ByeByeDPI-Manual)

### Как использовать ByeByeDPI вместе с AdGuard?
* Запустите ByeByeDPI в режиме прокси.
* Добавьте ByeByeDPI в исключения AdGuard на вкладке "Управление приложениями".
* В настройках AdGuard укажите прокси:
```plaintext
Тип прокси: SOCKS5
Хост: 127.0.0.1
Порт: 1080 (по умолчанию)
```

### Сборка
1. Клонируйте репозиторий с сабмодулями:
```bash
git clone --recurse-submodules
```
2. Запустите скрипт сборки из корня репозитория:
```bash
./gradlew assembleRelease
```
3. APK будет в `app/build/outputs/apk/release/`

> P.S.: hev_socks5_tunnel не соберется под Windows, вам нужно будет использовать WSL
